        加载外部音频资源 #

        本页内容：加载自定义语音、BGM 和音效。 前置知识：创建新剧本

        本 MOD 支持加载外部音频资源，支持 WAV（任意采样率/声道/位深度）和 OGG Vorbis 格式。

        在二创剧本中使用 #

        1. 创建音频目录 #

        在 MOD 目录下创建以下文件夹：

        • Voice/ — 角色对话语音
        • Audio/ — BGM 和音效
        [MOD根目录]/
  ├─ Voice/
  │   └─ TaffyStory000/
  │       └─ Noah_01.wav
  ├─ Audio/
  │   └─ Bgm_10_1.ogg
  ├─ Scripts/
  └─ info.json

        2. 在脚本中播放音频 #

        通过 @bgm 或 @voice 指令播放，文件名不含扩展名：

        ; 播放 BGM
@bgm "Bgm_10_1"

; 播放角色语音
@voice "TaffyStory000/Noah_01"

        替换游戏音频 #

        1. 获取原版音频编号 #

        可以通过解包工具 AssetRipper 解包音频资源查看对应的编号，也可以直接查看游戏自带的剧本寻找自己需要的音频编号。

        2. 覆盖原版音频 #

        在 Voice/ 或 Audio/ 目录下创建与游戏原音频相同路径和文件名的文件即可。MOD 音频的加载优先级高于游戏原版。
