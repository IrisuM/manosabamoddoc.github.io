        指令介绍 #

        back

        back #

        官方指令back

        设置背景元素

        参数说明
        subid指定层级，靠前的层级会遮挡靠后的，比如Middle会遮挡不设置时的默认层级
        @back SubId:"Middle" 1_1 pos:10,30 Scale:1.8
        bgm

        bgm #

        官方指令bgm

        设置背景音乐

        @bgm "Ambi_11_1"
        char

        char #

        官方指令char

        修改人物立绘

        参数说明
        subid指定层级，靠前的层级会遮挡靠后的，比如Middle会遮挡不设置时的默认层级
        @char SubId:"Middle" Warden.1 Scale:1.4
        gosub

        gosub #

        官方指令gosub

        跳转到指定路径，并在遇到 return 指令后返回

        goto

        goto #

        官方指令goto

        跳转到指定路径

        hide

        hide #

        官方指令hide

        隐藏指定的actor(角色, 背景, 文字框, 选择框)

        hidechars

        hidechars #

        官方指令hidechars

        隐藏所有可见的角色

        print

        print #

        官方指令print

        打印文本

        return

        return #

        官方指令return

        返回到gosub指令的调用处

        returntotitle

        returntotitle #

        游戏指令returntotitle

        返回标题页

        参数说明
        time动画时长
        delay延迟时间
        wait是否等待
        voice

        voice #

        官方指令voice

        播放指定角色对话声音

        wait

        wait #

        官方指令wait

        等待直到条件满足，条件通常为时间的浮点数
