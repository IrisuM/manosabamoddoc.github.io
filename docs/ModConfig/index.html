        Mod配置 #

        本Mod工具对于仅使用的人来说是开箱即用的，所有的设置选项都是面向需要编写剧本的人

        配置文件在 manosaba_game\BepInEx\config 目录下

        BepInex设置 #

        配置文件名为 BepInEx.cfg，这个是来自于BepInEx框架的配置

        如果你需要查看调试输出信息的话，需要将 [Logging.Console] 和 [Logging.Disk] 下的 LogLevels 配置改为All

        Mod设置 #

        配置文件名为 ManosabaLoader.cfg，这个是来自于本Mod框架的配置

        参数说明
        OpenDebug是否开启调试模式
        IsDirectMode是否跳过剧本选择菜单，直接进入预设入口
        ScriptEnter预设入口剧本文件
        ScriptEnterLabel预设入口剧本标签
        ModRootPath剧本根目录，不建议修改

        Debug模式说明 #

        当 OpenDebug 被设置为 true 时，将开启以下功能：

        1. 按下 ctrl + t 打开控制台。控制台为NaniNovel官方控制台，包括执行剧本指令、打开调试信息面板等功能

        2. 按下 ctrl + r 强制注销除了当前剧本外的剧本。当你修改剧本后发现重新进入剧本但修改没有生效时使用，建议在主菜单使用，避免出现非预期错误

        3. 转发游戏日志到BepInex的控制台

        4. 打印游戏尝试加载的资源路径
