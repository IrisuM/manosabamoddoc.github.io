        小贴士 #

        常见问题和编写技巧汇总。

        如何实时预览指令效果？

        打开 MOD 的调试模式，按下 Ctrl+T 可以开启控制台，在控制台中输入剧本指令即可实时执行查看效果。

        设置背景后人物不显示了？

        请关注 @back 和 @char 指令的 SubId 属性。如果背景层级在人物前面，就会遮挡人物。尽量保证在同一层级演出，不同层级用于场景过渡。

        导入外部音频出现加速或过慢？

        MOD 已支持任意采样率/声道的 WAV 和 OGG Vorbis 格式，正常情况下不应出现此问题。详见 加载外部音频资源。

        choice 指令只有第一个选项生效？

        这种问题大概率是缩进缺失。请检查缩进是否与官方剧本一致——一层缩进必须是 4 个空格！

        如何显示两个相同的角色？

        使用不同的 SubId 在不同图层显示角色即可。具体使用方法参考官方剧本。

        如何旋转人物？

        使用 @char 指令的 rotation 参数，参数类型为 x,y,z，其中 x、y、z 为角度值。

        如何制作双语剧本？

        参见 本地化配置。

        为什么不用官方的 BepInEx？

        截止开发插件完成时，官方的 BepInEx 与游戏本体不兼容，存在 GC 处理异常的问题，因此 MOD 安装包中附带了修复后的版本。
