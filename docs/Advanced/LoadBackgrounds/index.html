        加载外部背景资源 #

        本页内容：加载自定义背景图片或替换原版背景。 前置知识：创建新剧本

        本 MOD 支持加载外部背景资源，支持 JPG 和 PNG 格式图片。

        在二创剧本中使用 #

        1. 创建背景图片目录 #

        在 MOD 目录下创建 Backgrounds/MainBackground/ 文件夹，将需要添加的背景文件放在里面：

        [MOD根目录]/
  ├─ Backgrounds/
  │   └─ MainBackground/
  │       └─ TaffyCover.jpg
  ├─ Scripts/
  └─ info.json

        2. 通过 @back 指令加载背景 #

        使用 @back 指令 + 背景文件名（不含扩展名）即可：

        ; 设置背景为刚才添加的图片
@back TaffyCover

        替换游戏背景 #

        1. 获取原版背景编号 #

        可以通过解包工具 AssetRipper 解包背景图资源查看对应的编号，也可以直接查看游戏自带的剧本寻找自己需要的背景编号。

        游戏内背景路径由两部分组成：Id 和 图片名称编号，例如：

        @back 140_2 Id:"Stills" Scale:{g_backgroundDefaultScale} time:0
        • Id 为 Stills，图片名称编号为 140_2
        • 对应路径为 Backgrounds/Stills/140_2
        • 如果游戏没有指定 Id，则默认为 MainBackground

        2. 覆盖原版背景 #

        将二创图片的路径和文件名设置为与游戏原版一致即可。MOD 资源的加载优先级高于游戏原版。
