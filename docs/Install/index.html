        安装与卸载 #

        注意，使用Mod前记得保存原有的存档！
        虽然目前未发现损坏存档的情况，但是小心驶得万年船

        安装 #

        1. 下载安装包 #

          下载Mod安装包

        2. 安装文件 #

          打开游戏根目录

          将Mod安装包解压到游戏目录即可

        3. 完成 #

          最后启动游戏，初次启动或游戏版本更新后启动时间较久，请稍等

        卸载 #

        1. 删除文件 #

          删除游戏目录下被解压的文件即可
