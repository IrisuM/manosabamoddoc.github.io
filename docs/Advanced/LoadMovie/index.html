        视频播放 #

        本页内容：在剧本中播放自定义视频。 前置知识：创建新剧本

        添加视频文件 #

        将视频文件放置在 MOD 文件夹的 Movie/ 目录下：

        [MOD根目录]/
  ├─ Movie/
  │   ├─ Opening.mp4
  │   └─ Cutscene_01.webm
  ├─ Scripts/
  └─ info.json

        支持的视频格式：mp4、webm、ogv。

        在脚本中使用 #

        通过 @movie 指令播放视频（文件名不含扩展名）：

        ; 播放开场动画
@movie "Opening"

; 播放过场视频
@movie "Cutscene_01"

        视频播放期间玩家无法进行其他操作，播放结束后脚本自动继续执行。
