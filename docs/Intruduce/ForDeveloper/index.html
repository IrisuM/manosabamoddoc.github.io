        给想写剧本的人 #

        关于这个Mod框架 #

        《魔法少女的魔女裁判》是一款 Unity+NaniNovel 引擎开发的AVG游戏，而这个Mod框架则是基于BepInex实现对Unity引擎注入代码，将.nani剧本文件加载到NaniNovel引擎。

        因此本Mod框架加载的.nani脚本完全符合NaniNovel的官方标准，所使用的指令为官方支持的指令与游戏开发商自行开发的指令的合集。

        如果有一定的英语阅读能力，可以通过阅读NaniNovel的官方指引来快速掌握官方提供的基础引擎能力，而游戏开发商自行开发的指令，需要通过游戏内已有脚本反向推测功能。

        本文档也会继续完善部分指令的说明。

        想写剧本需要编程基础吗？会不会很难？ #

        完全不需要编程基础，NaniNovel官方在剧本编写上做了非常友好的用户体验设计，基本不会出现代码逻辑相关的操作，只需要记住一些用于操作对话框，人物表情，背景变化的指令即可

        为什么不能直接提供解包资源和游戏自带剧本？ #

        根据《魔法少女的魔女审判》直播・二次创作指南，不得超出二创范围使用拆包资源或者文字剧情，因此不提供任何现成的资源文件

        如果有二创需要，欢迎加入我们的群聊讨论，出于创作目的交流学习用的资源文件~
