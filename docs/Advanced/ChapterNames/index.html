        存档章节名称 #

        本页内容：自定义存档中的章节名称显示。 前置知识：创建新剧本

        info.json 配置 #

        在 info.json 中添加 ChapterNames 字段，指定每个脚本文件对应的章节名称：

        "ChapterNames": {
    "MaxMixAlex_ManosabaModEnhance/Main": "<voffset=-1em>梦中审判 序</voffset><br><size=80%>在梦中相遇的我们</size>",
    "MaxMixAlex_ManosabaModEnhance/Trial00": "<voffset=-1em>梦中审判</voffset><br><size=80%>希罗之罪</size>",
    "MaxMixAlex_ManosabaModEnhance/Ending": "<voffset=-1em>梦中审判 尾声</voffset><br><size=80%><color=#fe92b4>属于我们的时光</color></size>"
}

        键为 Naninovel 脚本路径 (与 Enter 字段使用相同的格式，如 ModId/Main)，值为显示的章节名称。

        支持富文本标签 #

        章节名称支持 Unity 富文本标签来自定义显示效果：

        标签说明示例
        <size=80%>调整字号<size=80%>副标题</size>
        <color=#ff0000>设置颜色<color=#fe92b4>粉色文字</color>
        <voffset=-1em>垂直偏移<voffset=-1em>上移文字</voffset>
        <br>换行主标题<br>副标题

        简单示例 #

        "ChapterNames": {
    "MyMod/Main": "第一章 序幕",
    "MyMod/Chapter1": "第一章 相遇",
    "MyMod/Trial": "魔女审判"
}
